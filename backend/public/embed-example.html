🤖 Askaksha AI Chatbot

Embeddable AI Assistant for Your Website

🚀 Quick Start

Add the Askaksha AI chatbot to your website in just 2 simple steps!

Step 1: Add the chatbot script to your HTML (before closing </body> tag)
<!-- Askaksha AI Chatbot --> <script src="https://your-backend-domain.com/embed-chatbot.js"></script> <script> AskakshaChat.init({ collegeCode: 'YOUR_COLLEGE_CODE' // Replace with your college code }); </script>
⚠️ Important: Replace YOUR_COLLEGE_CODE with your actual college code provided by Askaksha.

⚙️ Advanced Configuration

You can customize the chatbot with additional options:

<script> AskakshaChat.init({ collegeCode: 'YOUR_COLLEGE_CODE', // Required: Your college code serverUrl: 'https://your-backend-domain.com' // Optional: Custom backend URL }); </script>

Configuration Options

  • collegeCode (required): Your unique college identifier
  • serverUrl (optional): Custom server URL (default: script origin)

🔍 How It Works

  1. Load Script: The chatbot script loads from your Askaksha server
  2. Initialize: Call AskakshaChat.init() with your college code
  3. Floating Button: A chat button appears in the bottom-right corner
  4. User Interaction: Users click to open the chat window
  5. AI Responses: Messages are sent to your college-specific AI assistant
  6. Real-time Communication: Socket.IO ensures instant responses
